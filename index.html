Как да получа безплатен Gemini ключ?

Стъпка по стъпка

  1. Отворете Google AI Studio в браузъра:
    aistudio.google.com
  2. Влезте с вашия Google акаунт (същия, който ползвате за Gmail).
  3. В горния ляв ъгъл кликнете на "Get API Key" (Вземи API ключ).
  4. Натиснете "Create API Key" (Създай API ключ).
  5. Изберете съществуващ Google Cloud проект или оставете да се създаде нов автоматично.
  6. Ключът ще се появи на екрана — изглежда така: AIzaSy...
  7. Копирайте го и го поставете в полето "Gemini API Key" тук.
  8. Натиснете "Connect" и започнете разговор!

Важно да знаете

  1. Ключът е напълно безплатен — Google предоставя щедър безплатен лимит.
  2. Не споделяйте ключа с други хора — той е само за вас.
  3. Ако искате да спрете достъпа, можете по всяко време да изтриете ключа от AI Studio.
Поверителност и сигурност: Това приложение работи изцяло локално във вашия браузър. Няма сървър, няма база данни, нищо не се записва и не се споделя. Вашият API ключ се използва единствено за директна връзка между браузъра ви и Google Gemini — никой друг няма достъп до него.
